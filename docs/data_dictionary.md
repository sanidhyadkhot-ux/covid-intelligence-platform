# Data Dictionary

| Field | Description |
|---|---|
| location | Country name |
| continent | Continent |
| date | Reporting date |
| population | Country population |
| total_cases | Cumulative cases |
| total_deaths | Cumulative deaths |
| total_vaccinations | Cumulative vaccinations |
| infection_rate_pct | Total cases divided by population |
| case_fatality_rate_pct | Total deaths divided by total cases |
| vaccination_coverage_pct | Total vaccinations divided by population |
| risk_score | Demo composite risk score |
